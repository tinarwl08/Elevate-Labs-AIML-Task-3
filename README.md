Housing Price Prediction using Linear Regression
Project Overview
This project implements Simple and Multiple Linear Regression to predict housing prices using a
real-world dataset.
Objective
- Understand Linear Regression concepts
- Implement using Scikit-learn
- Evaluate performance using MAE, MSE, R²
Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
Dataset
Housing Price Prediction Dataset from Kaggle.
Steps Performed
1. Data Loading
2. Data Cleaning
3. Encoding Categorical Data
4. Train-Test Split
5. Model Training
6. Prediction
7. Evaluation
Model Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score
Results
The model successfully predicts housing prices with reasonable accuracy based on input features.
Conclusion
Linear Regression is a simple yet powerful algorithm for predictive modeling.
